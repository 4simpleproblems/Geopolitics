# Geopolitics

A minimalist, high-fidelity global strategy simulation and territorial builder.

## Features

- **Global Domination**: Annex every sector on the planet through strategic takeovers.
- **Dynamic Build Mode**: Create your own nations from scratch.
- **Auto-Coast Tracing**: Intelligent boundary plotting that snaps to real-world coastlines.
- **Sector Merging**: Combine existing territories into massive unified empires.
- **Tactical Strikes**: Execute strategic strikes with visual impact.
- **Minimalist Interface**: Clean, dark-ops aesthetic with high-performance rendering.

## Technology Stack

- **Globe.gl**: High-performance WebGL globe visualization.
- **Turf.js**: Advanced geospatial analysis and geometry manipulation.
- **Manrope**: Modern geometric typography.

## Quick Start

1. Open `index.html` in a modern browser.
2. Enter your starting nation.
3. Use the **Build Mode** to reshape the world or **Initiate Takeover** to expand your empire.
